# 永豐自動下單零股 (Auto-Order) v3.0

這是一個專為永豐金證券（Sinotrade）批次下單網頁設計的 **自動下單零股 TamperMonkey 腳本**。

經過多個版本的疊代與安全強化，3.0 正式版本結合了高質感介面、極致的操作效率，以及金融交易級別的安全防呆機制。

## ✨ v3.0 亮點特色
- **🛡️ 嚴格的下單前防護**：送出表單前深度驗證網頁的「股票名稱」與「數量」，若與指令檔不符或發現超過 999 股的異常大單，將主動攔截並彈出確認警告，確保您的遠端交易萬無一失。
- **🎯 100% 精確匹配**：獨家掃描所有下拉式節點內容，精確比對「代號」，避免從自動選單點選到名稱相似卻不相干的個股。
- **🤖 智慧動態等待**：網路順暢時瞬間捕捉下拉選單點擊；網路卡頓時提供最長 3.5 秒的容錯緩衝空間，讓每一檔股票的下單延遲降到最低卻不漏單。
- **🔒 原型鏈防護**：繞過券商前端框架（React/Vue）的安全鎖，確保原生輸入事件安全執行。
- **📱 現代化果凍面板**：極簡高質感的控制面板具備背景漸變 (Glassmorphism)，內建細節操作日誌（Log）與動態進度條。

## 🚀 系統需求
- 瀏覽器：Google Chrome / Edge
- 插件：TamperMonkey (油猴)

## 📦 如何安裝
1. 先安裝好 [Tampermonkey](https://www.tampermonkey.net/) 擴充功能。
2. 點擊安裝根目錄中的 `自動下單3.0.user.js` 將其加入腳本庫。（如果有舊版本，建議先移除停用以免衝突）
3. 進入永豐批次下單網頁，等頁面載入完畢，右上角就會自動浮出專屬面板。

## 📖 使用說明
- **貼上代號**：將今日想下單的零股代碼直接貼到輸入框中（支援連續換行、空格與逗號，非常自由）。
- **確認防呆**：建議維持勾選「異常時彈出確認框」，多一層安全網。
- **一鍵開始**：按下面板的「▶ 開始下單」，放開雙手，腳本即會自動幫您將所有的股票以「盤中零股」、「漲停價」送進待下單批次列當中！
